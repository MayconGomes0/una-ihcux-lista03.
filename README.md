Heurística de Jakob Nielsen (1994): "O sistema deve sempre manter os usuários informados sobre o que está acontecendo, através de feedback apropriado e na linguagem do usuário, com um tempo de resposta ideal de no mais 1 segundo."


Este projeto cumpre integralmente a 1ª Heurística de Nielsen, garantindo excelente visibilidade do status do sistema e feedback contínuo ao usuário.
